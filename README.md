# React Mini Apps (Feedback, Phonebook & Gallery)

## Описание

Это набор мини-приложений на **React**, объединённых на одной странице.  
Проект демонстрирует работу с состоянием компонентов, модальными окнами,
условным рендерингом и взаимодействием между компонентами.

Пользователь может:

- Оставлять обратную связь (Good / Neutral / Bad) и видеть статистику.
- Добавлять контакты в телефонную книгу.
- Фильтровать контакты по имени, фамилии или номеру телефона.
- Удалять контакты и добавлять их в избранное.
- Просматривать изображения в галерее.
- Открывать изображение в модальном окне для просмотра.

Приложение использует **Class Components**, переиспользуемые компоненты и
модальную систему.

---

# Реализовано

## Feedback Widget

- Кнопки для оценки: **Good / Neutral / Bad**.
- Подсчёт общего количества отзывов.
- Расчёт процента положительных отзывов.
- Сообщение, если отзывов пока нет.

---

## Phonebook

- Форма добавления контакта с полями:
  - Имя
  - Фамилия
  - Телефон
  - Пол (Male / Female)

- Валидация номера телефона через `react-phone-number-input`.
- Проверка на дублирование контактов:
  - по номеру телефона
  - по комбинации имени и фамилии.

- Возможности управления контактами:
  - удаление контакта
  - добавление в избранное
  - отображение аватара в зависимости от пола.

- Фильтрация контактов:
  - по имени
  - по фамилии
  - по номеру телефона.

- Если список контактов или избранного пуст — отображается уведомление.

---

## Gallery

- Отображение списка изображений.
- Возможность открыть изображение в **модальном окне**.
- Просмотр изображения в увеличенном размере.
- Передача данных изображения в компонент предпросмотра.

---

## Modal System

В приложении реализована переиспользуемая система модальных окон.

Используются два модальных компонента:

- **AboutAppModal**
  - описание всех мини-приложений
  - открывается из блока Hero

- **ImagePreviewModal**
  - отображает выбранное изображение из галереи
  - получает данные изображения через props

# Технологии

- **React 19** — построение пользовательского интерфейса
- **Vite** — сборщик проекта
- **PropTypes** — проверка типов props
- **react-icons** — иконки интерфейса
- **react-phone-number-input** — ввод и валидация телефонных номеров
- **Emotion** — CSS-in-JS стилизация
- **gh-pages** — деплой на GitHub Pages
- **react-toastify** — уведомления об успехах и ошибках

# Установка и запуск

## 1. Клонировать репозиторий

```bash
git clone <URL-репозитория>
cd react-mini-apps
```

## 2. Установить зависимости

```bash
npm install
```

## 3. Запуск проекта в режиме разработки

```bash
npm run dev
```

## 4. Сборка проекта

```bash
npm run build
```

## 5. Предпросмотр production сборки

```bash
npm run preview
```

## 6. Деплой на GitHub Pages

```bash
npm run deploy
```
